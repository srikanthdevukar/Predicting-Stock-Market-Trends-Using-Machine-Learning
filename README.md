# Predicting Stock Market Trends Using Machine Learning

A machine learning-based stock market prediction system that uses **LSTM**, **KNN**, and **Linear Regression** to forecast stock trends from historical market data.

## Project Overview
This project predicts stock market trends using an ensemble approach.  
It collects historical stock data, preprocesses it, trains multiple models, and compares their performance.

## Features
- Historical stock data collection
- Data preprocessing and normalization
- Stock trend prediction using:
  - LSTM
  - KNN
  - Linear Regression
- Graphical visualization of predictions
- Web-based interface using Django

## Tech Stack
- Python
- Django
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / Keras
- Yahoo Finance / Alpha Vantage

## How It Works
1. Collect stock data
2. Clean and preprocess the data
3. Train prediction models
4. Compare predicted and actual values
5. Display results on the web interface
